# Cuddly Happiness

## Project Setup Instructions (English)
1. Clone the repository: `git clone https://github.com/waleed14pc/cuddly-happiness.git`
2. Navigate to the project directory: `cd cuddly-happiness`
3. Install dependencies: `npm install` or `pip install -r requirements.txt` (depending on the project type)

## Usage Instructions (English)
Run the application using the command: `npm start` or `python main.py`.


# سعادة محبوبة

## تعليمات إعداد المشروع (العربية)
1. استنساخ المستودع: `git clone https://github.com/waleed14pc/cuddly-happiness.git`
2. الانتقال إلى دليل المشروع: `cd cuddly-happiness`
3. تثبيت التبعيات: `npm install` أو `pip install -r requirements.txt` (اعتماداً على نوع المشروع)

## تعليمات الاستخدام (العربية)
قم بتشغيل التطبيق باستخدام الأمر: `npm start` أو `python main.py`.
